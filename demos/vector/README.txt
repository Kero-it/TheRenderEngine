Vector Drawing Demo
-------------------

The vector demo is intended to show the vector drawing capabilities of The Render Engine.  It is an implementation of Asteroids (originally copyrighted by Atari, Corp. in 1979) with updated effects.  It demonstrates the following:

* Vector drawing with the vector component
* Physical motion with the 2D movement component
* Reading the query string for optional configurations
* The vector text renderer
* Keyboard input
* The particle engine & custom particle types
* The spatial grid collision model
* Convex hull collisions using SAT
* Sound loading and playback
* Image loading and display
* Reading and writing persistent local storage
* Custom configurations
